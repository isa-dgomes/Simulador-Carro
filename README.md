# Simulador-Carro

Esse projeto faz parte do meu andamento no bootcamp de Java que estou realizando pela plataforma DIO.

Aqui, esse projeto é um dos que compoem a etapa de aprendizado de Encapsulamento e Abstração.

Nesse simulador, o foco é bem simples, tendo interatividade via terminal, com foco nas regras de negócios específicas<br>
propostas pelo instrutor do curso no exercício.

O programa funciona apenas selecionando as opçoes que gostaria de realizar e assim, o programa retorna as funções.
